# Get-Domain 
 
Gets all available domains or the specified domain. 
 
## Syntax 
 
Get-Domain [-Name &lt;String&gt;] 
 
 
## Detailed Description 
 
The Get-Domain cmdlet returns all the domains or the specified domain.
