# Get-Archive 
 
Returns Sitecore database archives. 
 
## Syntax 
 
Get-Archive [[-Database] &lt;Database&gt;] [[-Name] &lt;String&gt;] 
 
 
## Detailed Description 
 
The Get-Archive command returns Sitecore archives in context of the specified database.
