# Get-ItemClone 
 
Returns clones of item provided. 
 
## Syntax 
 
Get-ItemClone [-Item] &lt;Item&gt; 
 
Get-ItemClone [-Path] &lt;String&gt; 
 
Get-ItemClone -Id &lt;String&gt; [-Database &lt;Database&gt;] 
 
 
## Detailed Description 
 
Gets all clones of item provided.
